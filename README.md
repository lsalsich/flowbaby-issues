# Flowbaby Issues

Welcome to the **public issue tracker** for [Flowbaby](https://flowbaby.ai) ‚Äî the AI-powered memory and context management system for VS Code.

Flowbaby helps developers maintain perfect context across coding sessions by automatically capturing decisions, progress, and reasoning, then making that knowledge retrievable through natural language. This repository is where you can report bugs, request features, ask questions, and engage with the Flowbaby community.

> **Note**: The core Flowbaby implementation is maintained in a private repository. This public repo is dedicated to community interaction, issue tracking, and roadmap visibility.

## üîí Privacy & Security

**Please do not share** the following in public issues or discussions:

- Security vulnerabilities (see below)
- API keys, tokens, or credentials
- Internal URLs or hostnames
- Customer or personal data
- Detailed deployment or infrastructure information

**Reporting security issues**: If you discover a security vulnerability, please email **security@flowbaby.ai** instead of opening a public issue. We take security seriously and will respond promptly.

See [CONTRIBUTING.md](CONTRIBUTING.md) for more details on our privacy and security policies.
